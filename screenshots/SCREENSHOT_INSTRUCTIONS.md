# Admin Dashboard Screenshot Instructions

## Setup

1. Install **GoFullPage** Chrome extension:
   https://chrome.google.com/webstore/detail/gofullpage-full-page-scre/fdpohaocaechififmbbbbbknoalclacl

2. Make sure you're logged into the admin at http://localhost:3000/admin

## How to Use GoFullPage

1. Navigate to the page
2. Wait for it to fully load
3. Click the GoFullPage extension icon (camera icon)
4. Wait for capture to complete
5. Click the download icon (PNG format)
6. Rename the file and save to `screenshots/ezcr-admin/`

## Tips

- Wait 2-3 seconds after page load before capturing
- For pages with lazy-loaded content, scroll down first to trigger loading
- Some pages may have loading spinners - wait for them to complete
